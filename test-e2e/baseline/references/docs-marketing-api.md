Marketing API

The Marketing API is a collection of 
Graph API
 endpoints and other features that can be used to help you advertise across Meta technologies. Before advertising on Facebook, Instagram, Messenger, and WhatsApp, we recommend you learn about Meta&#039;s 
ad campaign structure
 to understand the objects you&#039;re working with and how they relate to each other.

The latest version is
:

v24.0

For information on the latest Marketing API version, see the 
Changelog
. For non-versioned changes, see the 
Out-of-Cycle Changes
.
